3. Just DESPRÉS de l'overlay (encara abans de `</body>`), afegir el bloc de script:
```html
  <script>
    document.querySelector('form').addEventListener('submit', function () {
      var btn = this.querySelector('button[type="submit"]');
      var overlay = document.getElementById('loading-overlay');
      btn.disabled = true;
      btn.textContent = 'Processant...';
      overlay.hidden = false;
    });

    window.addEventListener('pageshow', function (e) {
      if (e.persisted) {
        document.getElementById('loading-overlay').hidden = true;
        var btn = document.querySelector('button[type="submit"]');
        if (btn) {
          btn.disabled = false;
          btn.textContent = 'Corregir article';
        }
      }
    });
  </script>
```

IMPORTANT: NO afegir e.preventDefault() al submit handler — la petició POST ha de continuar normalment. L'overlay es mostra mentre el navegador espera la resposta del servidor.

Per crear el favicon.ico: generar un fitxer ICO mínim de 32x32 píxels de color sòlid #1a3a5c usant Python i el mòdul `struct` (sense Pillow). Si Pillow és disponible, usar-lo. Desar a web/static/favicon.ico.

Script Python per generar favicon.ico sense dependències externes (ICO format 32x32 BMP):
```python
import struct

def make_ico(path):
    # BMP header for 32x32 RGBA (4 bytes/pixel)
    w, h = 32, 32
    r, g, b = 0x1a, 0x3a, 0x5c  # #1a3a5c
    pixels = bytes([b, g, r, 255] * w * h)  # BGRA order

    # DIB header (BITMAPINFOHEADER) for ICO
    bmp_header = struct.pack('<IIIHHIIIIII',
        40,       # biSize
        w,        # biWidth
        h * 2,    # biHeight (doubled for ICO: XOR + AND mask)
        1,        # biPlanes
        32,       # biBitCount
        0,        # biCompression (BI_RGB)
        len(pixels),  # biSizeImage
        0, 0, 0, 0    # resolution, colors
    )

    # AND mask (all zeros = fully opaque)
    and_mask = bytes(h * ((w + 31) // 32) * 4)

    image_data = bmp_header + pixels + and_mask

    # ICO directory header
    ico_header = struct.pack('<HHH', 0, 1, 1)  # reserved, type=1 (ICO), count=1
    ico_entry = struct.pack('<BBBBHHII',
        w, h, 0, 0,  # width, height, colorCount, reserved
        1, 32,        # planes, bitCount
        len(image_data),  # sizeInBytes
        6 + 16        # offset (6 header + 16 entry)
    )

    with open(path, 'wb') as f:
        f.write(ico_header + ico_entry + image_data)

make_ico('web/static/favicon.ico')
```

Executar aquest script des del directori arrel del tema (on hi ha `web/`). Alternativament, si Pillow és disponible: `from PIL import Image; img = Image.new('RGB', (32,32), (0x1a,0x3a,0x5c)); img.save('web/static/favicon.ico')`.
